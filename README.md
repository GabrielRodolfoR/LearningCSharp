## Requisitos

- .NET SDK 8.0 ou superior instalado em sua máquina.
- Um editor de código compatível, como Visual Studio Code ou Visual Studio 2022.
- Abra o arquivo Program.cs e rode ele ou utilize o comando:

  ```bash
  dotnet run
  ```
